QR Code Embedding (Option 2)

Using the qr-embed.js helper to turn <img> tags into working QR codes.

This looks like an image but is a functional QR!

How to use:

1. Add the qr-api class to your image tags.

2. Link to the qr-embed.js script at the bottom of your page.

<img class="qr-api" src="api/index.html?data=Hello" width="200" height="200" />
<script src="lib/qr-embed.js"></script>
